- full page

- 원래는 시멘틱 구조를 시멘틱 태그로 이용해서 웹을 만들려고 했는데,
욕심이 생겨서 중간에 fullpage를 만들어 보았다.
머리속에 생각하는 그림이 있는데 그걸 사용하려면 풀페이지 기능이 필요하다.
물론 예전에 구매한 fullpage.js를 사용하면 바로 되지만 그건 내 실력으로 만드는게
아니라 라이브러리를 사용해서 만드는 것 이기 때문에 다른것은 몰라도 나의 포트폴리오를
만드는데는 라이브러리를 사용을 자제 할려고한다.

오늘은 완벽하지는 않지만 어느정도 움직이는 것 까지는 대충 구현이 되었고,
첫 움직임과 마지막 페이지 일때 움직임의 코드가 부족한것 이외에는 대충 기능을
어떻게 구현할지 만들었다.

(23.06.13)
- full page

- 오늘은 거의 완성 단계이다. 어제 움직이는 부분을 보안하고, 첫 움직임과 마지막 페이지 일때 움직임
이 두개의 오류를 해결을 했는데, 살짝 부족하다.
두번째 페이지 부터는 transitionend를 사용할 수가 있다. 왜냐면
첫번쨰 페이지를 움직여서 이벤트가 발생이 됐는데 첫번째 페이지는 transitionend가 발생이 안된다.
그래서 첫번째와 마지막 번째 페이지 Vs 그 중간의 페이지들의 넘기는 속도 차이가 난다.
그래서 내일은 이 부분을 보안하면 완성이 될 것같다.
